## How to use
- Options is array of AccordionOption objects

```js
export type AccordionOption =
  label?: string;
  child?: React.ReactNode;
  text?: string;
;
```
- For each option you can specify
  - label - text for radio button
  - child - components which renders if radio is selected
  - text - text bellow radio button
