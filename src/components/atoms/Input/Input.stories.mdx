# Input Component

Straithforward input component. It can be simple, or with placeholder.
When an error occurs its border changes color

## Notes

- Default error value is set to false, so if omitted the field will render without error border

- With onChange prop you can set the state to the inputField from parent components.

- onnHandleCapslook prop is used for the password molecule, to inform component when caps-lock is pressed.
