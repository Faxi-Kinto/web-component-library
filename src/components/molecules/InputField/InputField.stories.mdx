# InpitField Component
This field is a wrapper around [Input](/docs/core-atoms-input--default-story) atom and it is intended for use in [Form Field](/docs/core-molecules-formfield--default-story)
enabeling error messages, error state for input atom and description message for the field.
InputField implements FormField API.
