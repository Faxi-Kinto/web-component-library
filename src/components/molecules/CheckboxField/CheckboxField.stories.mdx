# CheckboxField Component

This field is a wrapper around [Checkbox](/docs/core-atoms-checkbox--default-story) atom and it is intended for use in [Form Field](/docs/core-molecules-formfield--default-story)
enabeling error messages, error state for input atom and description message for the field.
`CheckboxField` implements FormField API.

#### Default

<Preview>
  <Story name="default" decorators={[withKnobs]}>
    <CheckboxField
      label="Click me!"
      error={text('Error message', 'This field is required')}
      description={text('Description message', '')}
      labelPosition={(boolean('Label on left',false)) ? 'left':'right'}
      touched={boolean('Touched', false)}
      dirty={boolean('Dirty', false)}
