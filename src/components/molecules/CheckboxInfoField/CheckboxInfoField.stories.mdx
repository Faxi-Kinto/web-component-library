# CheckboxInfoField Component

This field is a wrapper around [Checkbox](/docs/core-atoms-checkbox--default-story) atom and it is intended for use in [Form Field](/docs/core-molecules-formfield--default-story)
as part of "Terms and Conditions" form enabeling: error messages, error state for input atom and description message for the field.
`CheckboxInfoField` implements FormField API and it has minor styling/functionality changes in relation to `CheckboxField`

`label` and `description` are mandatory.

`lastElement` propery options out the bottom border line
